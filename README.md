# Лабораторные работы по прикладным системам и фреймворкам искусственного интеллекта

## Описание

В рамках данного репозитория представлены результаты выполнения 5 лабораторных работ, посвященных исследованию различных алгоритмов машинного обучения на датасете, взятом с [kaggle](https://www.kaggle.com/datasets/akshaydattatraykhare/diabetes-dataset). Для каждой лабораторной работы реализованы базовые модели из sklearn, их улучшенные версии с подбором гиперпараметров и препроцессингом данных, а также собственные реализации алгоритмов. Исследования проводятся для задач классификации (предсказание наличия диабета) и регрессии (предсказание возраста пациентов).
